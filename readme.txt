== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'mj-news' file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[mjnews]` in your page editor or in templates paste this code mj_news();
